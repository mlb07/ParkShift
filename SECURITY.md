# Security

## Supported Versions

ParkShift is pre-1.0. Security fixes are made on the main development line.

## Reporting A Vulnerability

Please do not post secrets, exploit details, or private data in public issues.

For now, report security concerns privately to the repository owner. If this project becomes public and shared more broadly, replace this section with a dedicated security contact.

## Sensitive Data

ParkShift should not require API keys or private credentials. Do not commit:

- `.env` files
- API tokens
- private datasets
- downloaded bulk data
- trained model artifacts that include private data
